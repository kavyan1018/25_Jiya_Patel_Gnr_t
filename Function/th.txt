what is Function in c ?
--> in c programming, a function is a block of code that performs a specific task. It can take inputs (arguments), process them, and return an output (result). Functions help in organizing code, making it reusable and easier to read.

types of functions in C:
    1. NR - NA -> No Return, No Arguments
    2. NR - WA -> No Return, With Arguments
    3. WR - NA -> With Return, No Arguments
    4. WR - WA -> With Return, With Arguments
